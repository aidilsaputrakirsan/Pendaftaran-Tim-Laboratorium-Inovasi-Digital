Detail Posisi

System & Infrastructure Engineer

Jobdesk
  • Melakukan setup dan konfigurasi VPS dan cPanel
  • Mengelola keamanan server dan backup system
  • Mengoptimasi kinerja infrastruktur
  • Melakukan pemeliharaan dan monitoring server
  • Membangun pipeline untuk continuous integration/deployment
  • Melakukan audit keamanan dan testing secara berkala
  • Menangani troubleshooting teknis
Kualifikasi
  • Dasar administrasi Linux (atau memiliki kemauan kuat untuk mempelajarinya)
  • Familiar dengan konsep cloud server/VPS (atau bersedia belajar secara intensif)
  • Memahami dasar-dasar jaringan dan keamanan sistem
  • Antusias mempelajari teknologi virtualisasi dan containerization
  • Memiliki mindset security-first dan keinginan untuk terus berkembang

Full-stack Developer

Jobdesk
  • Mengembangkan dashboard monitoring penggunaan lab
  • Membuat sistem manajemen user dan resource
  • Mengintegrasikan layanan-layanan yang diperlukan
  • Mengembangkan template deployment untuk berbagai use case
  • Membantu otomatisasi proses dan workflow
  • Membuat tools pendukung untuk kemudahan pengguna
  • Memberikan dukungan teknis untuk deployment aplikasi
Kualifikasi
  • Menguasai dasar pemrograman web (atau bersemangat untuk belajar)
  • Memahami konsep dasar database (dan berkomitmen memperdalam pengetahuan)
  • Familiar dengan minimal satu bahasa pemrograman
  • Kemauan belajar framework web modern
  • Antusiasme tinggi untuk mempelajari DevOps dan konsep CI/CD
  • Kemampuan problem-solving yang baik

Documentation & Community Manager

Jobdesk
  • Membuat dokumentasi penggunaan sistem untuk pengguna
  • Menyusun SOP dan panduan teknis
  • Mengembangkan konten tutorial dan pelatihan
  • Mengelola website/knowledge base lab
  • Membuat materi sosialisasi dan promosi lab
  • Menangani dukungan pengguna (user support)
  • Mengorganisir kegiatan lab (workshop, pelatihan, dll)
  • Mengumpulkan feedback dari pengguna untuk perbaikan
Kualifikasi
  • Kemampuan menulis yang baik
  • Kemampuan komunikasi dan presentasi
  • Kemauan belajar dasar-dasar teknologi yang digunakan
  • Semangat untuk mengembangkan diri dalam hal manajemen konten
  • Memiliki jiwa customer service dan problem-solving
  • Minat belajar dalam dokumentasi teknis dan manajemen komunitas
